# Learning Spring

Interactive course app for Java, Kotlin, the JVM, JPA, and Spring fundamentals.

## Runtime

- Node.js 25.9.0

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Lessons

- Java entry point and `main`
- Kotlin data classes
- JVM references and memory model basics
- Spring constructor injection
- Controller to service flow
- JPA entity identity
- Transaction boundaries
- Focused Spring testing

## Workspace

- Kotlin Playground: https://play.kotlinlang.org/
